# Grupo SysBR - Site Corporativo

Site institucional do Grupo SysBR, desenvolvido com base na apresentação corporativa da empresa.

## 🚀 Sobre o Projeto

Site moderno e responsivo que apresenta os serviços, diferenciais e verticais de atuação do Grupo SysBR, uma holding com mais de 25 anos de experiência no mercado de tecnologia.

## 📋 Características

- Design moderno e responsivo
- Animações suaves e interativas
- Menu de navegação mobile
- Seções destacando:
  - Diferenciais da empresa
  - Plataforma de desenvolvimento própria
  - Verticais de atuação (Varejo, Aplicativos, Shopping, Tributária, Contábil, Saúde)
  - Números e estatísticas
  - Formulário de contato

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e animações)
- JavaScript (Vanilla)
- Google Fonts (Inter)

## 📁 Estrutura de Arquivos

```
SiteGrupoSys/
│
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # JavaScript para interatividade
└── README.md          # Este arquivo
```

## 🌐 Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Navegue pelas diferentes seções usando o menu superior
3. O site é totalmente responsivo e funciona em dispositivos móveis

## 📱 Responsividade

O site se adapta automaticamente a diferentes tamanhos de tela:
- Desktop (>968px)
- Tablet (768px - 968px)
- Mobile (<768px)

## 🎨 Personalização

### Cores

As cores principais estão definidas em variáveis CSS no arquivo `styles.css`:

```css
--primary-color: #0066ff;
--secondary-color: #00d4ff;
--dark-color: #1a1a2e;
--light-color: #f5f7fa;
```

### Fontes

O site utiliza a fonte **Inter** do Google Fonts. Para alterar, modifique o link no `<head>` do HTML.
